# 介绍

所有的模块均在 @liuli-util 组织下发布，请查看 https://www.npmjs.com/org/liuli-utilopen in new window

  • DEV
    • create-cli: 用于创建 cli/lib 工具的脚手架
    • vite-plugin-node: 使用 vite 构建 nodejs 程序需要的一切
    • rollup-plugin-i18next-dts-genopen in new window: 从国际化文件中生成 d.ts 文件
  • LIB
    • tree: 树结构操作实用函数库
    • react-router: 封装 react-router 为集中式的 js 路由配置，组件仅暴露必要的 props，并且默认支持在 react 组件外使用路由。
